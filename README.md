🌸 ACGNHUB

ACGNHUB 是一个以 ACG（Anime / Comic / Game / Novel）内容为核心的个人博客网站，专注于动漫、漫画、游戏与轻小说相关内容的整理与分享。

网站通过文章形式，对各类 ACG 资源、网站入口、工具以及相关内容进行整理，同时结合计算机与软件教程，打造一个兼具兴趣与实用性的内容平台。

🚀 部署说明

本项目为基于 WordPress 的博客网站，可部署于任意支持 LNMP/LAMP 环境的服务器。

基础部署流程：

1. 准备服务器（Linux 推荐）


2. 安装 Nginx / MySQL / PHP


3. 部署 WordPress


4. 配置域名解析


5. 安装并配置主题（Sakurairo）


6. 初始化网站结构与分类
