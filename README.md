# 🌸 ACGNHUB

**ACGNHUB** 是一个专注于 ACG（Anime / Comic / Game / Novel）领域的资源整理与导航网站，致力于为用户提供高质量的二次元网站入口与内容推荐。

项目目标是构建一个 **简洁、高效、易用的 ACG 资源索引平台**，帮助用户快速找到优质的动漫、漫画、游戏及相关社区资源。

# 🛠 技术栈

本项目基于以下技术构建：

### 后端

* WordPress

### 服务器环境

* Nginx
* MySQL
* PHP 8.2

### 部署环境

* 阿里云 ECS（Ubuntu）
* 宝塔面板（BT Panel）

### 前端

* HTML / CSS / JavaScript（主题与页面）

---

# 🚀 部署说明

本项目为 WordPress 网站，可部署于任意支持 LNMP/LAMP 环境的服务器。

### 基础部署流程：

1. 准备服务器（推荐 Linux）
2. 安装 Nginx / MySQL / PHP
3. 上传并安装 WordPress
4. 配置域名解析
5. 安装主题（Sakurairo）
6. 完成站点初始化
