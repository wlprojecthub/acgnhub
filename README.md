ACGNHUB

ACGNHUB 是一个轻量级的 ACG（Anime / Comic / Game / Novel）导航网站，用于整理和收集常用的二次元相关网站资源。

项目的目标是打造一个 简洁、快速、易用的 ACG 网站入口导航，帮助用户快速访问常用的动漫、漫画、游戏及社区网站。

---
